SIMPLE DESIGN SYSTEM


Untuk perkenalan singkat apa itu design system, biasanya di artikan sebagai kumpulan komponen desain yang nantinya akan digunakan secara berulang-ulang. Keberadaan design system ini sendiri sangatlah penting untuk menjaga konsistensi dari desain keseluruhan serta mempermudah desainer ataupun developer ketika akan melakukan perubahan/penambahan pada desain yang telah ada.

Ada 3 plugin yang saya gunakan untuk mempermudah saya dalam membuat design system ini :

  • Styler, berguna untuk meng-generate color atau font ke local styles di file figma
  • Foundation: Color Generator, plugin yang membantu dalam membuat color token maupun color palettes
  • Redlines, plugin yang dapat membantu desainer dan developer untuk mengetahui jarak/space antar komponen desain

Berikut simple design system yang saya buat berdasarkan hasil UI Exploration yang sebelumnya pernah saya buat yaitu, landing page C-RAFT - Furniture Landing Page.




Dari proses yang saya lalui, saya belajar bahwa salah satu ciri dari design system yang baik adalah design system yang dapat dimengerti bukan hanya oleh designer saja tetapi dapat juga dipahami oleh role lainnya, seperti developer, ketika nantinya desain akan diimplementasikan ke dalam bentuk kodingan.

Ke depannya saya akan selalu meng-update dan menambah design system ini, sehingga tidak hanya terbatas pada color hingga button saja.
